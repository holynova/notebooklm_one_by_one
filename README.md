# NotebookLM Auto Slide 自动化生成工具

这是一个基于 Node.js 编写的懒人批量生成工具。只需提供一批 URL，即可让 NotebookLM 帮您自动创建新的 Notebook、导入来源、并批量生出漂亮的幻灯片 (Slide Deck)！

## 🌟 特点
1. **自动化环境与登录检查**：全自动检测依赖与认证情况，支持一键帮你安装和登录授权。
2. **支持批量 URL 投喂**：支持换行、空格、逗号等方式一次性丢入多个 URL。
3. **智能化与防刷机制**：自动生成匿名 Notebook 供官方重命名，内置随机化安全延迟防止接口报错。
4. **带有详细的执行日志**：原生记录每一步 CLI 发送的命令以及输出的日志，方便排错。

## 📦 快速开始

### 运行总入口
```bash
node main.js
```

### 执行效果
1. **阶段一：环境检查 (`check_env.js`)**
   - 检查 `nlm` 是否安装（若未安装将自动调用 uv 帮你安装）。
   - 检查 `nlm` 是否已授权您的 Google 账号（未登录会自动打开浏览器授权）。
2. **阶段二：执行任务 (`auto_slide.js`)**
   - 弹出交互式界面，在此不断粘贴您的所有目标 URL。
   - 输入您对 Slide 的总结提示词 (Focus)。如果您这留空，系统将根据内容自动发挥。
   - 自动发车执行批量生成。

## ⚙️ 依赖项
- [Node.js](https://nodejs.org/) (确保已安装即可)
- [uv](https://docs.astral.sh/uv/) (Python 包管理工具，如果没有，需按照官方手动安装一次 uv。如果不使用此脚本的自动安装，也可以自己手动 `uv tool install notebooklm-mcp-cli`)
- [notebooklm-mcp-cli](https://github.com/jacob-bd/notebooklm-mcp-cli) (本工具自动为您集成管理)

## ⚠️ 常见问答
- **遇到报错并跳出？**
  本工具由于采用了完整的错误阻断。如果遇到未登录、超时或是 CLI 的问题，为了保证安全它会直接退出进程。您可以根据控制台输出的原始 CLI `<<< 错误信息` 解决后再继续。
